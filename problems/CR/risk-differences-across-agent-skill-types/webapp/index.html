Methodology

Dataset: 3,500 AI agent skill definitions across 7 functional categories (500 per category), sourced from public tool registries and agent platforms.

Analysis: Each skill was analyzed for 8 vulnerability types using static analysis, pattern matching, and manual expert review. Severity scored on CVSS-aligned 1-10 scale.

Statistical Methods: Chi-square omnibus test for overall distribution differences. Pearson and Spearman correlations for permission-prevalence relationship. Risk Disparity Index computed as max/min prevalence ratio.

Composite Score: 0.6 * normalized_prevalence + 0.4 * normalized_severity, providing balanced risk ranking across categories.
