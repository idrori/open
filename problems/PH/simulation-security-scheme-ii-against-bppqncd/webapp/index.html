Problem Overview

Conjecture 5.1 (Stambler, 2026)

The one-time memory Scheme II is simulation-secure in the quantum random oracle model against adversaries in the class BPP^QNC^d for any polynomial depth d. The scheme uses conjugate coding (BB84 encoding) combined with hash-locking via a random oracle and conjunction obfuscation to hide the encoding basis.
